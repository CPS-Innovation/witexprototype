Latest updates

The latest updates, improvements, and bug fixes for our service.

Tasks

8 Aug 2024 Introduction of ‘Priority Charging’ tab

We have added a new tab to show all of the priority charging tasks in one place. This view will show you when the request was received and the due date and time (which is three hours after the request was received).

If a user has the ‘View Charging tasks – National’ right, then they will see the priority charging tasks nationally, as their default view. If a user does not have this right, then they can filter by Units to see the relevant priority tasks.

Please note: we are aware that the due date is calculated wrong at 4 hours and not 3 hours. We will release a fix for this as soon as possible.

8 Aug 2024 ‘Activate Tasks’ now available:

You will now be able to activate tasks from the task list. There will be a button in the expanded view of a task where there will be three possible outcomes depending on the type of task. These are:

  • when you select ‘Start task’ the CMS Classic/Modern tab will change to the appropriate screen for that case (similar to how Modern changes Classic)
  • when you select ‘Mark as Complete’ the App will display a pop-up enabling you to mark the task as complete and add any relevant notes
  • when you select ‘Start task’ the CMS Classic/Modern tab will change to the appropriate screen for that case and the Casework App will appear in another tab

Please note: the first time you activate one of these tasks – 28 day PCD Review, 5 day PCD review, Check New PCD Case, Priority PCD Review, you will get a ‘pop-up blocked’.

Please do the following –

  • click on ‘Start Task’
  • you will see in the right-hand side of the URL a ‘Pop-up blocked’ message (in red below)
  • when you select ‘Start task’ the CMS Classic/Modern tab will change to the appropriate screen for that case and the Casework App will appear in another tab
  • click on this and you will get the pop-up shown in red:
  • click ‘Done’
  • click on ‘Start Task’ again and then the casework app will open in a new tab and CMS Classic will be changed to the appropriate screen for that task
  • you will not get this message again unless updates are made to Windows

Please note: as new Future Casework Tools are built, we will update the tasks with new links to take you to the appropriate place for that task. For example, in the future, we will take you to the correct case within the new FCT Review tool.

If you click on the URN of a task, the CMS Classic tab will change to the case details screen for that case.

8 Aug 2024 Unit groups

A new Unit Group filter has been added enabling users to search for tasks assigned to the units in the group selected.

8 Aug 2024 Updates to the Directions tab

We have introduced the CTL icons to the directions table to help users prioritise.
